# Roadmap

Phases, not dates. Cloud credit approval and timing are outside our
control, so we're not attaching commitments we can't keep.

## Phase 0 — Foundation (current)
- Public website and project foundation live at tre-up.com
- Product constitution and behavioral design principles locked internally
- Cloud credit applications in progress (AWS, Google Cloud, Microsoft
  Azure)
- Early-access interest collection

## Phase 1 — Prototype / MVP
- Backend API foundation
- Secure signal storage and data pipeline
- First iOS client shell (onboarding, core screens)

## Phase 2 — Hawk Scoring Prototype
- First working version of the signal-to-guidance layer
- Internal testing of prediction quality and confidence handling

## Phase 3 — Private Beta
- Small group of early users, invitation-based
- Direct feedback loop on whether Hawk's guidance is actually useful

## Phase 4 — Premium V1 (iOS)
- Full iOS experience, subscription model
- Public availability

No dates are attached to these phases yet. They'll be added once Phase 1
infrastructure is actually in place — not before.
